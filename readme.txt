== Description ==

SKT Software is a multipurpose WordPress theme which can be used for IT, repair, services, online medium, coaching, corporate, business, program, freeware, application, operating system, laptop, computer, courseware, productivity, file management and others. It comes with ready to import 63+ Elementor templates which can be used for home and inner pages. It is fast, flexible, simple and fully customizable. It is Gutenberg friendly and works with Siteorigin, Beaver builder, Brizy  too.WooCommerce compatible.
